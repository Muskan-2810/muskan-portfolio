# Muskan Vishwakarma — Portfolio

A cinematic, 3D, AI-themed developer portfolio built with Next.js App Router,
React Three Fiber, GSAP, and Framer Motion.

## Stack
- Next.js 14 (App Router) + TypeScript
- TailwindCSS (custom "deep space" token system)
- React Three Fiber + Drei (galaxy/starfield background, upcoming 3D robot & neural network scenes)
- GSAP (page loader, hero text reveal, scroll-triggered timelines)
- Framer Motion (magnetic buttons, micro-interactions)
- Lenis (smooth scroll)

## Getting started

```bash
npm install
npm run dev
```

Then open http://localhost:3000.
